# Dra. Ivanilde Cavalcanti - Site Oficial

Site da Dra. Ivanilde Cavalcanti, Endocrinologista e Metabologista.

## Páginas

- **index.html** — Home
- **sobre.html** — Sobre a Dra. Ivanilde
- **especialidades.html** — Áreas de Atuação (9 especialidades)
- **clinica.html** — A Clínica

## Tecnologias

- HTML5 + TailwindCSS (CDN)
- Google Fonts (Newsreader)
- Material Symbols

## Como rodar localmente

```bash
python3 -m http.server 3000
# Acesse: http://localhost:3000
```
